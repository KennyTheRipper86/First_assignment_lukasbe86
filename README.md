# Einfacher Gewohnheits-Tracker (Assignment 1)

Dieses Programm ist ein minimalistischer Tracker, der dabei hilft, tägliche Gewohnheiten aufzubauen und zu verfolgen. Es zählt Erfolgsserien (Streaks) für jede Gewohnheit, um die Motivation zu fördern.

Um das Programm zu starten, führe die Python-Datei `lukas_behlau_assignment1.py` in einem Terminal aus: `python lukas_behlau_assignment1.py`.

Die Besonderheit dieses Programms ist, dass es bewusst nur Python-Grundlagen verwendet. Um einen neuen Tag zu simulieren und die Gewohnheiten zurückzusetzen, muss die Menü-Option "Neuer Tag" gewählt werden. Alle Daten sind nur für die Dauer der Programmsitzung gespeichert.
